# SEO & Keyword Engineering

This skill equips the agent with advanced strategies to discover the exact terms required to find niche visual assets and reference materials.

## Core Concepts

1. **Search Intent Deconstruction**: Break down what the user is *actually* asking for. If a user asks for "happy people working," the intent translates to "authentic corporate lifestyle photography."
2. **Boolean Search & Operators**: Use standard and advanced search operators (AND, OR, NOT, "exact phrase", site:pinterest.com, filetype:png) to filter out noise.
3. **Keyword Expansion (The 4 Tiers)**:
   - *Broad*: Generic terms (e.g., "dotted background")
   - *Niche*: Specific styles (e.g., "pop art comic style dots")
   - *Long-Tail*: Hyper-specific queries (e.g., "seamless retro halftone dot matrix pattern illustration")
   - *Industry-Jargon*: Professional terminology (e.g., "stippling," "Ben-Day dots," "screentone")

## Implementation Workflow

- **Input Analysis**: Review the user's brief.
- **Root Term Extraction**: Identify the noun/adjective pair that is central to the request.
- **Synonym & Jargon Mapping**: Expand root terms into professional design and media keywords.
- **Platform-Specific Optimization**: Adjust the keyword phrasing based on the target platform (e.g., Pinterest prefers aesthetic/mood keywords; Getty Images prefers literal, descriptive tags).
