# Visual Semiotics & Terminology

Clients and users often lack the vocabulary to describe the exact visual style they envision. This skill allows the agent to act as a translator between everyday language and professional design parlance.

## Core Translation Concepts

### 1. Pattern & Texture Translation
- "Dotted" -> Halftone, Ben-Day dots, stippling, screentone, polka dot, dot matrix.
- "Rough/Scratchy" -> Grunge, distressed, grain, noise, stipple, dry brush.
- "Shiny/Plastic" -> Glossy, glassmorphism, 3D render, subsurface scattering, chrome.

### 2. Layout & Structure Translation
- "Boxes everywhere" -> Bento grid, masonry layout, modular grid, card-based interface.
- "Clean and simple" -> Minimalist, whitespace, Swiss design, flat design, neo-brutalism (sometimes).

### 3. Color & Lighting Translation
- "Neon/Bright lights" -> Cyberpunk, synthwave, bioluminescence, rim lighting, duotone.
- "Old/Faded" -> Sepia, vintage, muted palette, film grain, light leak, lo-fi aesthetic.

## Implementation Workflow

1. **Extract Layperson Terms**: Identify non-technical descriptive words in the user's prompt (e.g., "I want it to look like a comic book").
2. **Map to Design History/Styles**: Connect the description to established art movements or design trends (e.g., Comic book -> Pop Art, Roy Lichtenstein, CMYK halftone print).
3. **Output Professional Terminology**: Provide the user with the exact industry terms to use for their search or prompt generation.
