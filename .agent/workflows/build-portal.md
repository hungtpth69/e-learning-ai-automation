1. Kích hoạt Python Compiler Lấy Data 
```bash
cd "e:\Antigravity\e-learning\Internal Portal (Vercel Deploy)"
python optimized_raw_input.py
```

2. Ra Lệnh Cho IDE Agent 🚀
Sau khi hệ thống cô đặc thành công file cấu trúc `data.js`, ở cửa sổ Terminal sẽ thả xuống 1 đoạn Câu Lệnh Lập Trình (Prompt).
👉 Học viên hãy **Sao Chép (Copy)** nguyên đoạn lệnh trong khung Nét Đứt.
👉 Dán vào Khung Chat Của Antigravity AI Assistant ở góc bên phải màn hình. Nhấn Enter.

Trợ lý thông minh AI sẽ tự động đọc `data.js` và tạo lập nguyên thủ công cho bạn một file `index.html` siêu đẹp ngay lập tức! Bạn chỉ việc mở lên chiêm ngưỡng.
